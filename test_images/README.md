# Test images

* [horse72x40.gif](horse72x40.gif) Black and white animation of a running horse in 72x40 pixels. Original createf by [Eadweard Muybridge](https://en.wikipedia.org/wiki/Eadweard_Muybridge) in 1878 by photografing a running horse with 11 cameras. For further information see [The Horse in Motion](https://en.wikipedia.org/wiki/The_Horse_in_Motion).
* [TreescapInterlaced.gif](TreescapInterlaced.gif) an example of an interlaced image.
